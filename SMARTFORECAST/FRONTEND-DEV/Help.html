How accurate are the forecasts?
Forecasts are typically 85–95% accurate for the next 3 months. Accuracy decreases for longer periods due to external factors.
Can I import data from my accounting software?
Yes! We support direct integrations with QuickBooks, Xero, and FreshBooks. Check the Integrations tab.
What happens if I exceed my budget?
You’ll receive an alert, but no actions are blocked. It’s just a warning to help you manage spending.
Is my data encrypted?
Yes, all data is encrypted both in transit (SSL/TLS) and at rest (AES-256). We comply with SOC 2 standards.
Can I export reports?
Yes, export reports as PDF or CSV from any dashboard page. Use the “Export” button in the top right.
What is the maximum file size for imports?
You can import files up to 100MB. For larger datasets, please contact support.
